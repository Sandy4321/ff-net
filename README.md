# ff-net
A feedforward neural network learning in real time. [See the live demo.](http://juniorrojas.github.io/ff-net)

![screenshot](media/screenshot.png)

## development

Install dependencies

```sh
npm install
```

Build and watch source for development

```sh
npm run dev
```

Publish to `gh-pages`

```sh
npm run build
npm run deploy
```
